Coin Bucket

UIUX Design Case Study - Money saving app

About this app

Coin bucket is designed to build saving habits and help to achieve their goal by creating multiple buckets. The app is designed to inspire the user to save on a regular basis. It also help people with less knowledge of finance.

Design Process

User Needs

When doing user research, recorgnise user needs by focusing on people’s goals rather than thier preferences.

User Flow

Based on research above, I created user flow That reflects progress from start to end.

Wireframes

I sketched out main screens wireframes and developed them into mid fidelity prototype.

User Testing & Feedback

Refined bucket Details

Initially the graph is having a scroll bar which is hard to understand, what is the use of that. So changed this by adding year at the top with backward and forward arrow. And give more weight to deposit button.

Refined New goals

Another issue faced by user is that they are not aware how many steps will be there in creating a goal. So I added a stepper for more clear view.

Colors

I choose one primary color for brand and rest colors are basic to keep the app simple.

Typography

Poppins is used throughout the app in various size and weight.

User dashboard

A dashboard that encourages them to save and creates habits. A chart of total savings that gives the user a positive feeling and a progress bar for each area gives the user an overview of the progress of their goals.

Bucket details

The bucket details section contains all the details about the user's target. The bucket details also have a deposit and withdrawal button. The bucket details screen has two views for the transaction to visualize the goals and their progress.

Wallet

The wallet screen displays the total amount with the credit card UI to give the user the feeling of having money. The user can also view their past transactions and the transactions are marked with +/- to indicate the type of transaction.

An application for manifesting your goals by building habits
